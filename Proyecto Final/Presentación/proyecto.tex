\section{Revisión de la Literatura}
\begin{frame}{Revisión de la Literatura}
	\begin{defi}
		\textbf{Clasificación de noticias falsas} se refiere al proceso de determinar cuando una noticia contiene información falsa/desinformativa o no.
	\end{defi}
	\textcolor{red}{\textbf{Enfoque Tradicional}}
	\begin{itemize}
		\item hecha por expertos y periodistas.
		\item problema: grandes volúmenes de información.
	\end{itemize}
\textcolor{blue}{\textbf{Enfoque Reciente}}
\begin{itemize}
	\item automatizar el proceso de clasificación.
	\item propuesta: usar aprendizaje máquina y  NLP para noticias y publicidades.
\end{itemize}
\end{frame}
%%%%%%%%%%%%%%%%  F R A M E  7  %%%%%%%%%%%%%%%%
\begin{frame}{Algunas definiciones}
	\begin{defi}
		\textbf{Noticia falsa}. Se refiere a información fabricada que imita el contenido de los medios de comunicación en forma pero no en el proceso organizativo e intencional.
	\end{defi}
	\textcolor{blue}{\textbf{Hipótesis de Underutsch}}
	\\
	La diferencia entre las noticias falsas y verdaderas está en términos de estilo de escritura (autenticidad, intención)
	\\
	\textcolor{blue}{\textbf{Teoría de los Cuatro Factores}}
	\\
	La  afirmaciones verdaderas falsas y verdaderas son expresadas con distintas emociones y sentimientos.
	\\
	\textcolor{white}{text}
	\\
	\textcolor{white}{text}
	\\
	\textcolor{white}{text}
	\\
	\textbf{Hipótesis} $ \to $ emociones y sentimientos ayudan a distinguir afirmaciones verdaderas y falsas.
\end{frame}

\section{Conclusiones}
\begin{frame}{Conclusiones}
		\begin{enumerate}
			\item Se extendió el dataset LIAR agregando emociones usando la API de IBM NLP y un puntaje de sentimiento usando la API de Google NLP.
			\item Tanto para los experimentos realizados con FFNN como con CNN se incrementó el acurracy del estado del arte para el dataset LIAR.
			\item El mejor modelo fue en el que se le dio TEXT directamente a BERT y luego se concatenó con el resto de los atributos para entrenar la parte de CNN.
			\item El modelo tuvo un mejor resultado al usar los atributos EMO Y SEN, justo como lo decía la hipótesis de Undeutsch.
		\end{enumerate}
\end{frame}
